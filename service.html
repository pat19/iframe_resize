常見問題

我無法成功啟動視訊客服？

請確定您的裝置有對應的視訊設備：

您可以從磁帶將視訊會入圍 Windows Media Video ( WMV ) 檔，或使用數位攝影機 ( DV ) 預設格式的檔案，例如：音訊視訊交錯格式 ( AVI ) 檔。您也可以選擇「匯入視訊」對磁帶上的每個剪輯建立WMV檔。例如，如果磁帶尚有八個剪輯，「匯入視訊」可以建立八個獨立的 WMV 檔。
